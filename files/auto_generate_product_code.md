The code of products/items can be autogenerated. \
You can change this from settings. If Auto Generate is ON, then **IMS** will use the **Code prefix** \+ **Category Code Prefix** + **Auto Generated Number** to generate unique code

\
Follow below steps
 - Go to Settings 
 - ![](https://app.imsdesk.com/knowledgebase/images/go_to_settings.png)
 - Undert section Items, enable **Auto Generate**
 - ![](https://app.imsdesk.com/knowledgebase/images/settings_items_auto_gen.png)
  - You can set a code prefix value also if needed, which will be used as prefix for all items under any category 
 - Also inside category, there is a **Code Prefix** field. If there is a value, it'll also be used to generate the code.
 - ![](https://app.imsdesk.com/knowledgebase/images/category_code_prefix_field.png)
