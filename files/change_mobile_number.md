**You can change your phone number from Organization editor**

\
Click the **Change Mobile Number** icon.

You need to verify using a OTP. 
 
\
Steps:

1. Go to navigation menu
2. Click Organization
3. From Organization editor, click on the **Change Mobile Number** icon at top right
4. ![](https://app.imsdesk.com/knowledgebase/images/change_mobile_number.png)

> This feature is currently available in web version only
