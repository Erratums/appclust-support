
If you want your user to allow to set previous bill date then you need to enable this from settings. 

 - Go to settings 
 - ![](https://app.imsdesk.com/knowledgebase/images/go_to_settings.png)
 - In Invoice and Purchase sections, you can see a tickbox allow Previous Date. 
 - **Enable it** to allow users to set previous date in invoice and purchase respectively
- ![](https://app.imsdesk.com/knowledgebase/images/settings_allow_users_to_set_previous_date.png)
