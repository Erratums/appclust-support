**It's very easy to create an account. We provide sign in with Google option also**

\
**Follow below steps** 

  1. Go to [IMS Web App](https://app.imsdesk.com)
  1. You will see a login screen as below 
  1. ![](https://app.imsdesk.com/knowledgebase/images/login_screen.png)
  1. If you already have an account, please login 
  1. If you don't have an account, you can use **Sign in with Google** which will automatically create an account for you with the selected email address.
> Please note that, the admin user should have a Google Account
