**You can add more modules to home screen tab list**

\
by default the home page will show only 4 tabs [Invoice, Purchase, Stock and Items], if you want, you can include other tabs here so that it's easy to access. 

> In mobile devices, you can only have a maximum of 4 tabs

To add more tabs,   
1. Go to Settings 
2. Then click on **Visible Modules** under **General Settings**
3. ![](https://app.imsdesk.com/knowledgebase/images/settings_visible_modules.png)
4. Check all the modules which you need in the **Home Page**
5. ![](https://app.imsdesk.com/knowledgebase/images/settings_visible_modules_popup.png)
6. This is a local settings, so if you need these tabs in another device or computer, you need to do the sames settings from that device/computer also
